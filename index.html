Hnefatafl Q-learning

Author: Nick Pnevmatikatos

A project that applies Q-learning to Hnefatafl (a historical board game) to train agents and evaluate learned strategies.

Overview

This repository contains the implementation, experiments, and visualisations for an RL agent trained with Q-learning to play Copenhagen Hnefatafl. The core goals:

  • Implement game mechanics and state representation.
  • Train agents via Q-learning and evaluate performance.
  • Document results and present findings.
